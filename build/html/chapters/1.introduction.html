Introduction¶

The N&F-Trivia app is a platform for playing trivia in concert with a video call. It runs on an express webserver and performs dynamic updates the contestants pages. The host can customize the questions which are stored with a Google account.

To create and host a game visit https://frar.ca/trivia/host.ejs
